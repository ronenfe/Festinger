Downloads




    Download Historia Version 1.11.
    

    note: the program is not compatible with Windows Mobile Standard ( Smartphones ).

Registering

    To register, simply register to our forum ( you have a link on the top), and then send ronenfe a private message asking for the     key and giving your email, you will get the license key to your email for free in a short while.
   


Instructions


  1. If you don't have Windows Mobile 6, and don't have .NET Compact Framework 2.0, you have to download and install it first before you install Historia, you can download it from here:                                                                               .NET Compact Framework 2.0 SP2( Microsoft Site)                                                                                                          or from here :                                                                                                                                                      .NET Compact Framework 2.0 SP2( this site ).
  2. Download Historia from the link on the top. Unzip the file, and run Setup.exe , don't copy it to the device, it will be installed with Activesync, follow the on screen instructions.
  3. You can run the program 5 times before having to register. tap Register Later on the logon screen.
  4. If you registered to our forum and recevied a key, enter your email address and key number and tap Register on the logon screen.
